심사 방법 안내

1
작품 선택
왼쪽에서 분야(아동청소년부/청년부) 선택 후, 목록에서 작품명을 클릭하면 심사 화면으로 이동합니다.
2
작품 이미지 확인
이미지를 클릭하면 확대됩니다. 확대된 화면에서 이미지 옆 빈 공간을 클릭하면 다시 축소됩니다.
3
점수 입력
각 항목별로 1점(부적합) / 3점(미흡) / 5점(보통) / 7점(우수) / 10점(매우우수) 중 하나를 선택하세요. 7개 항목의 합계가 총점(최대 70점)으로 자동 계산됩니다.
4
저장
반드시 💾 저장 버튼을 클릭하세요. 저장 버튼은 화면 상단과 하단 모두 있습니다. 재접속 후에도 수정 가능합니다.
5
필터 및 검색
왼쪽에서 전체/심사완료/미심사 필터 선택, 또는 검색창에 작품명·번호 입력. 정렬은 번호순/점수 높은순/낮은순 선택 가능합니다.
